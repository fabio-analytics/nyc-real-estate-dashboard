# 🏙️ NYC Real Estate Analytics Dashboard
**Um projeto de Ciência de Dados por Vision Data Pro**

Este é um dashboard interativo desenvolvido em Python para análise de vendas no mercado imobiliário da cidade de Nova York. A aplicação permite explorar a distribuição de preços e tamanhos de imóveis, utilizando a API do Mapbox para renderização geoespacial de alta precisão.

## 🔧 Funcionalidades

- **Visão Histórica:** Apresenta todas as transações de imóveis em Nova York registradas no dataset.
- **Análise Espacial:** Mapeamento interativo de imóveis classificados por faixa de preço, ano de construção e localização.
- **Filtros Dinâmicos:** Segmentação de imóveis a partir do seu tamanho (metragem quadrada) e região (Borough) com atualização em tempo real.
